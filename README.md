# VVU-DeskShell — лабораторная работа №4 (БОС)

Учебная мини-оболочка на Python (tkinter).

## Запуск

```powershell
cd "путь\к\лаба 4"
py -3 main.py
```

Тестовые пользователи: `vadim` / `vadim123`, `student` / `bos2026`

## Возможности

- Авторизация и регистрация (пароль — PBKDF2-SHA256)
- Калькулятор, файловый менеджер, диспетчер задач, параметры
- Смена темы, фона и обоев (`wallpapers/`)
- Значки на рабочем столе (`icons/`)

## Структура

| Файл / папка | Описание |
|--------------|----------|
| `main.py` | Основной код оболочки |
| `users.json` | Учётные записи (хеши паролей) |
| `icons/` | PNG-иконки приложений |
| `wallpapers/` | Обои рабочего стола |
